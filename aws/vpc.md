1. What is AWS VPC?
    - VPC ( Virtual Private Cloud ) is Logically isolated Virtual Network
    - Contains Subnet
    - Components/Features
        - Subnets
        - IP Addressing
        - Routing
        - Gateways and Endpoints
        - Peering Connections
        - Traffic Mirroring
        - Transit Gateways
        - VPC Flow logs
        - VPN Connections
